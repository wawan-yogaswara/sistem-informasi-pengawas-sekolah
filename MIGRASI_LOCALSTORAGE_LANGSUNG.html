üöÄ Langkah 2: Migrasi ke Production

Instruksi:

  1. Setelah menjalankan script ekstrak di localhost:5000
  2. Copy script yang dihasilkan dari console
  3. Buka aplikasi production di browser
  4. Buka Developer Console (F12)
  5. Paste script migrasi dan tekan Enter
üìù Catatan:
Script migrasi akan otomatis dihasilkan setelah Anda menjalankan script ekstrak di localhost:5000. Script tersebut sudah dioptimasi untuk menghindari masalah "canceled".
